== Installation ==

1. Faça upload da pasta `auto-post-ai` para o diretório `/wp-content/plugins/`.
2. Ative o plugin no menu 'Plugins' do WordPress.
3. Configure sua API Key da OpenAI em Auto Post AI > Configurações.
